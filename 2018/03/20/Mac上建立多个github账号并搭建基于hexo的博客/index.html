###起

一直想建立一个独立而隐秘博客站点，记录过往的故事心情，终于下了决心做，第一篇就成了技术文。参考了很多网上资料，对于同一台电脑建立多个 github 账号和 搭建基于 Hexo 博客的教程不少，但是没有找到两者统一的。因为不想和平时工作的github 账号混淆，所以需要再添加一个新的github账号，并用新账号写博客，就自己稍稍探索了一下，方法如下。
